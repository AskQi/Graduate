# IGES文件解析与三维可视化程序设计
## 说明
毕业设计源代码
## 要求及内容

初始图形交换规范(IGES)是一种设计与制造过程的几何形状描述标准，在CAD建模、CAE仿真、CAM制造过程中均由广泛应用。本课题要求采用MATLAB或其它编程语言对IGES文件进行解析，得到曲线、曲面几何形状信息，并实现其三维可视化。
